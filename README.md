# IdeaFlow Marketplace

Claude Code plugin marketplace from [IdeaFlow](https://github.com/IdeaFlowCo).

## Install

```bash
/plugin marketplace add IdeaFlowCo/ideaflow-marketplace
/plugin install ideaflow-skills
```

## Plugins

### ideaflow-skills

Skills for iOS app publishing and Claude skill development.

| Skill | Description |
|-------|-------------|
| `ios-app-store-launch` | Launch an iOS app on the App Store from Xcode project to live listing via CLI and App Store Connect API |
| `skill-crafting-guide` | Step-by-step guide for building high-quality skills for Claude — built by synthesizing [Anthropic's official guide](https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf), production skill analysis, and community best practices |

## Usage

Once installed, skills are available as:

- `/ideaflow-skills:ios-app-store-launch`
- `/ideaflow-skills:skill-crafting-guide`

Or they trigger automatically when you ask Claude to submit an app to the App Store or create a skill.
